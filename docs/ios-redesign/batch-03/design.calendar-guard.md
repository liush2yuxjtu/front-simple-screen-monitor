# Design · Calendar Guard

Version name: `calendar-guard`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 活页夹、边界、安静工作日。
- Best for: Focus、Meeting、Travel 这类日程保护。
- Risk: 会更像 office product。

## Layout

- 顶部是 binder tabs，不是警戒条。
- 当前保护逻辑被写成 cover sheet。
- “别打断今天”被做成安静的整理动作。

## Why Choose It

- 最像真实工作工具。
- 能把边界感做得成熟而不吵。
