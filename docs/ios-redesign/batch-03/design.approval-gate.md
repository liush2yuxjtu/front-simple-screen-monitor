# Design · Approval Gate

Version name: `approval-gate`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 覆纸、许可、轻边界。
- Best for: 会切焦点、改状态、跨 app 的动作。
- Risk: 如果覆层太重，会像系统弹窗。

## Layout

- 主动作先露出，再被一张 vellum check sheet 轻盖住。
- 许可条件被压成覆纸语气，而不是门禁语气。
- 胜出方向的亲和感优先保留。

## Why Choose It

- 最能把 consent 做得自然。
- 适合后续接系统边界和权限设计。
