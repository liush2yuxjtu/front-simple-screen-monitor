# Design · Yolo Sprint

Version name: `yolo-sprint`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 急件纸条、前插、轻快。
- Best for: 低风险、强时效、可逆动作。
- Risk: 阈值放错会显得抢。

## Layout

- 主动作像斜插在 agenda 里的 rush slip。
- 三层 slip 解释 why now、scope、boundary。
- 速度感来自纸条前置，不来自重动画。

## Why Choose It

- 最能测试 winner 能否承载更主动默认态。
- 仍然比“快控制台”更像产品。
