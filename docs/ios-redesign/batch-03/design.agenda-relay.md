# Design · Agenda Relay

Version name: `agenda-relay`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: ribbon、接力、顺手。
- Best for: 会议前后和连续排程动作。
- Risk: 会更坚定地把产品拉向 agenda 助手。

## Layout

- now、next、later 被串成一条 ribbon。
- 动作像沿着日程被传给下一个时间点。
- 比旧版更少“卡片竞争感”，更多“时间流感”。

## Why Choose It

- 是最顺手、最自然的一支。
- 如果你喜欢 batch 02 winner 的本味，这版最稳。
