# Design · Moss Triad

Version name: `moss-triad`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 三联账页、白天、轻策略。
- Best for: 把胜出方向做成新的总 thesis。
- Risk: 如果模式定义太虚，会失去三联结构的价值。

## Layout

- 三模式像三张并排纸条，而不是一个孤立 selector。
- agenda spine 在上，动作卡在中，三联解释卡在下。
- 主体始终先像排程助手，再像 agent 策略层。

## Why Choose It

- 最适合当 rebatch 后的新基准。
- 它把胜出方向推进了，但没有背离它。
