# Design · Safe Harbor

Version name: `safe-harbor`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 证据架、稳、安静。
- Best for: 需要先看依据的中风险动作。
- Risk: 如果证据块太多，会拖慢节奏。

## Layout

- 证据被做成两张 proof cards，而不是厚说明文。
- 主动作退回建议层，依据站到桌面前排。
- 仍保留胜出方向的白和轻。

## Why Choose It

- 最适合把“稳”做得像产品，而不是后台。
- 能验证轻 UI 是否也能承载保守策略。
