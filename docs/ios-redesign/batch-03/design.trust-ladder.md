# Design · Trust Ladder

Version name: `trust-ladder`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 页边批注、信任轨、出版感。
- Best for: 解释 agent 为什么停在当前档位。
- Risk: 如果页边内容太多，仍会显教条。

## Layout

- trust steps 被挪到页边 rail。
- 主卡重新回到动作本身，不再背着整块教学区。
- 像杂志页边批注，不像设置面板。

## Why Choose It

- 最适合做长期 trust teaching。
- 解释性很强，但主页面貌更轻了。
