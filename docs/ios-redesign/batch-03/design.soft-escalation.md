# Design · Soft Escalation

Version name: `soft-escalation`
Status: `REBATCH_OPTION_BATCH_03_2026-04-28`

## Core

- Mood: 嵌套纸片、渐进、策略感。
- Best for: 想把风险升级做得可见又不技术化。
- Risk: 如果规则不清，纸片层级会变装饰。

## Layout

- 三个档位被做成逐层内收的 step cards。
- 风险升高时，纸片往中心靠，而不是忽然跳色报警。
- 仍然站在 winner 的纸面和明亮气质里。

## Why Choose It

- 最有产品后劲。
- 很适合从 UI 往真实 policy 继续长。
