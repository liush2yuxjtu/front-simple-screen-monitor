# Design · Oat Calendar

Version name: `oat-calendar`
Status: `DRAFT_OPTION_BATCH_02_2026-04-28`

## Core

- Mood: 生活日程、柔和、家庭组织。
- Best for: 出门、接人、买菜、会议前准备等生活任务。
- Risk: 太亲和时会削弱专业产品感。

## Palette

- Background: oat `#efe2cf`
- Surface: cream `#fbf4e8`
- Accent: oat olive `#9a9c63`
- Support: toast brown `#58493f`

## Layout

- 顶部像今日日期条。
- 主卡强调“今天这一步”，不是抽象意图。
- detail 更像日程补注。

## Motion

- 滑动像在今日清单里划过。
- 执行后出现温和完成态。
- 稍后更像顺延。

## SwiftUI Mapping

- `HeaderView` 可加入日期和日程条。
- `ActionProposalCard` 更适合家务与生活行动。
- 长期使用压力最低。

## Why Choose It

- 最适合生活型 use cases。
- 能把 Action Stream 变成真正可用的日常组织器。
