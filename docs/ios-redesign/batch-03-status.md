document: swipev2 iOS redesign batch 03 is
`MOSS_AGENDA_REBATCH_OF_8_READY_2026-04-28`, with 8 local HTML concept boards

## What Changed

- Rebatched Batch 03 around the winning qualities of `Moss Agenda`.
- Kept the daytime, low-pressure, agenda-led product feeling.
- Replaced the older “8 similar mode strategy boards” direction with 8 more
  material, paper-like, memorable variants.
- Added the Batch 00 current iOS app baseline into the same comparison loop.
- Extended the version-switch loop to include the Batch 01 winner
  `Sand Signal` and the Batch 02 winner `Moss Agenda`, so the comparison flow
  is now `1 current app + 2 winners + 8 rebatch variants`.
- Kept left/right version switching on the index page and keyboard navigation
  inside every design HTML.

  variant changes.
- Confirm the loop now includes `Current iOS App`, `Sand Signal`,
  `Moss Agenda`, and the 8 Batch 03 rebatch pages for a total of 11 switch
  targets.
- Open the 8 design HTML pages and confirm they still feel rooted in the Batch
  02 winner, not like a dark console or alert-heavy system.

  tri-fold, rush slip, proof shelf, vellum sheet, ribbon relay, margin rail,
